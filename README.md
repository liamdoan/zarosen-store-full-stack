# zarosen-store-full-stack
The deployment of the project is on Heroku and can be found via this link: https://zarosen-clothing-store.herokuapp.com/

User side: Users can create their own account, log in and log out of the account, add/clear products to/from wishlist and paylist.

Server side: admin of the website can 
- manage user accounts' information
- add and modify products. Similar products will not be created in the products database

More features for the website are in process and will be updated soon
